Límites en Matemáticas

Recurso educativo interactivo – Segundo de Bachillerato

¿Qué es un límite?

Un límite describe el valor al que se aproxima una función cuando la variable independiente se acerca a un número determinado. Es fundamental para entender el comportamiento de funciones y el cambio.

Concepto Intuitivo de Límite

El límite se entiende como un proceso de aproximación. Aunque el valor no se alcance, la función puede acercarse cada vez más a un número.

Notación del Límite

La expresión lim (x → a) f(x) = L significa que f(x) se aproxima a L cuando x se acerca a a.

Límites Laterales

Un límite puede analizarse por la izquierda (x → a⁻) y por la derecha (x → a⁺). Si ambos coinciden, el límite existe.

Cálculo de Límites Directos

Cuando la función es continua, el límite se calcula sustituyendo directamente el valor.

Ejemplo: lim (x → 2) (x² + 3x) = 10

Indeterminación 0 / 0

Cuando al sustituir obtenemos 0/0, se debe simplificar la expresión algebraica.

Interpretación Gráfica
